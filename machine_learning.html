📘 Machine Learning Full Theory

Machine Learning is a branch of Artificial Intelligence where computers learn from data.

🧠 How Machine Learning Works

  • Data Collection
  • Data Cleaning
  • Model Training
  • Testing
  • Prediction

Output = f(Data + Algorithm)

⚡ Types of Machine Learning

Supervised Learning

Uses labeled data like spam detection.

Unsupervised Learning

Finds patterns in unlabeled data.

Reinforcement Learning

Learns from reward and punishment.

🌍 Applications

  • Face Recognition
  • Self Driving Cars
  • Chatbots
  • Medical Diagnosis
  • Recommendation Systems

🔥 Advanced Concepts

Overfitting = model learns too much and fails on new data.

Underfitting = model learns too little.
